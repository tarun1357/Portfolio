# Tarun Saini — Portfolio

Personal portfolio site (Next.js App Router, TypeScript, Tailwind CSS v4).

## Prerequisites

- Node.js 18.18+ (recommended: current LTS) with **npm** installed

## Setup

```bash
cd tarun-portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command        | Description              |
| -------------- | ------------------------ |
| `npm run dev`  | Dev server (Turbopack)   |
| `npm run build`| Production build         |
| `npm run start`| Run production server    |
| `npm run lint` | ESLint                   |
| `npm run format` | Prettier               |

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Motion:** Framer Motion (dependency installed; wire up in sections next)
- **Lint/format:** ESLint + Prettier + `eslint-config-prettier`

## Deployment

Designed for **Vercel**: connect the repo and deploy; same build as `npm run build`.
